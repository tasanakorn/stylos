# Origin of Stylos

Stylos originated as the foundation for a zenoh-based interconnect layer and is now maintained here as its primary repository.

## Current repo position

This repository should be treated as the **primary repository for Stylos**.

That means:

- core Stylos docs live here
- future crate-level evolution should be documented here
- downstream consumers should treat this repo as the source of truth for current Stylos library behavior

## Migration note

Some historical design context may still exist in older project history elsewhere, but current documentation should be maintained in this repository.
