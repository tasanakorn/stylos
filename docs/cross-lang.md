# Stylos Cross-Language Notes

Stylos is designed around zenoh, so multiple language bindings can participate as long as they share compatible transport and key-expression conventions.

## Status

| Language | Status |
| --- | --- |
| Rust | primary implementation in this repo |
| Go | possible follow-on support |
| Python | plausible future target |
| TypeScript | possible future target depending on binding capabilities |

## Rust

This repository is the primary home for the Rust foundation crates.

## Go

Go participation is a reasonable future direction for Stylos. When that support is added here, this document should describe the setup and prerequisites directly in this repository.
